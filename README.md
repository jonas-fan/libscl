# libscl
[![Build Status](https://travis-ci.org/yjfan/libscl.svg?branch=master)](https://travis-ci.org/yjfan/libscl)

The simple container library for C

## Build
```
$ git clone https://github.com/yjfan/libscl.git
$ cd libscl/
$ mkdir build/
$ cd build/
$ cmake ..
$ make -j4
$ make install
```

## Container
- List
- Vector
- Queue (List based)
- Stack (List based)
- Ring buffer
